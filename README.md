# 📚 School Management System

A simple and interactive School Management System developed using **Python**, **Object-Oriented Programming (OOP)**, **JSON File Handling**, and **Streamlit**.

This project allows users to manage student and teacher records, assign grades, and maintain school data through a user-friendly interface.

---

## 🚀 Features

### 👨‍🎓 Student Management

* Register new students
* Store student details
* View all student records

### 👩‍🏫 Teacher Management

* Register teachers
* Store teacher information
* View teacher records

### 📝 Grade Management

* Record student grades
* Update academic performance
* Maintain structured grade data

### 💾 Data Persistence

* Data is stored in JSON format
* Automatic loading and saving of records

## ▶️ How to Run

### Clone the Repository

```bash
git clone https://github.com/your-username/School-Management-System.git
cd School-Management-System
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the Application

```bash
streamlit run app.py
```
